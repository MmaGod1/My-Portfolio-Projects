# Portfolio Projects

## Skills and Concepts Practiced

While building these projects, I practiced and strengthened the following concepts:

- Asynchronous JavaScript (handling promises and async operations)
- Array of objects (data structuring and manipulation)  - DOM API (manipulating and interacting with web pages)
- Events (event handling and delegation)
- Font Awesome (icon integration)
- navigator.clipboard.writeText() (clipboard / copy-to-clipboard functionality)
- HTML5 Drag and Drop API (dragstart, dragover, drop events)
- dataTransfer API (passing data between draggable elements and drop zones)
- DOM manipulation using appendChild() (moving elements between containers)
